# Claude Entry Point

This file is read automatically by Claude Code at the start of each session.

Before making project changes:

1. Read `.ai/instructions.md`.
2. Follow the workflow it specifies for the current task.

Keep durable project context in `.ai/`, not in session memory.

## Figma MCP

The Figma MCP server is available in this environment. Use it when the user
asks about Figma files, designs, or Make projects — but always read
`.ai/instructions.md` first so you understand the process constraints before
touching anything.
