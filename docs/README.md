# PuzzleFactory Design System Docs

This directory contains human-readable design-system guidance, architecture notes, and ADRs.

These docs are discoverable reference material. They are not mandatory startup reading for every agent session unless the current task touches the topic.

## About The Packages

- [Color Engine](./about/color-engine.md): specialized color primitive, semantic color, diagnostic, and CSS artifact generation.
- [Tokens](./about/tokens.md): stable broader design-token model and future non-color token generation.
- [Themes](./about/themes.md): full theme composition across color, typography, density, elevation, component defaults, and artifact metadata.
- [Theme Authoring](./about/theme-authoring.md): designer-facing normalized theme input, preview, diagnostics, and artifact export workflow.
- [Components](./about/components.md): component consumption boundaries for semantic CSS custom properties.
